## ✨ Features

- 📝 Add new tasks with title and description  
- 📊 Organize tasks into:
  - To Do  
  - In Progress  
  - Done  
- 🔄 Drag & Drop tasks between columns  
- ❌ Delete tasks  
- 💾 Persistent data using LocalStorage  
- 🔢 Real-time task count per column  
- 🎯 Simple and clean UI
